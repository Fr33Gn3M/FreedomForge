## 1. 技术架构

```
                      ┌─────────────────────┐
                      │   前端 (Naive UI)    │
                      │  行情面板 + 策略管理   │
                      │  回测报告 + 信号中心   │
                      └──────────┬──────────┘
                                 │ REST API / WebSocket
                      ┌──────────▼──────────┐
                      │  后端 (FastAPI)      │
                      │  /api/quant/*       │
                      └──────────┬──────────┘
              ┌──────────────────┼──────────────────┐
     ┌────────▼────────┐  ┌──────▼──────┐  ┌────────▼────────┐
     │  market_data.py │  │ strategy.py │  │ signal_engine.py │
     │  行情拉取+存储    │  │ 策略加载+运行 │  │  信号生成+风控    │
     └────────┬────────┘  └──────┬──────┘  └────────┬────────┘
              │                  │                   │
     ┌────────▼────────┐  ┌──────▼──────┐  ┌────────▼────────┐
     │  SQLite/CSV     │  │ backtrader  │  │ OKX Agent        │
     │  历史K线数据库    │  │ 回测引擎     │  │ Trade Kit (MCP)  │
     └─────────────────┘  └─────────────┘  └─────────────────┘
```

**数据流向：**
1. OKX API → market_data.py → SQLite（历史数据积累）
2. 用户策略文件 → strategy.py → backtrader（回测）
3. 实时行情 + 策略 → signal_engine.py → 前端展示
4. 信号审核通过 → OKX MCP → 下单

**推荐技术栈：**

| 层 | 技术 | 为什么 |
|----|------|--------|
| 行情数据 | `okx-trade-cli` 或 `ccxt` | OKX CLI 封装完善，ccxt 支持多交易所 |
| 历史存储 | SQLite Pandas | 轻量无部署，DataFrame 直接读写 |
| 回测引擎 | `backtrader` | 文档全、社区大、策略模板丰富 |
| K 线图 | `ECharts` | Naive UI 原生集成，支持暗色主题 |
| 执行 | OKX Agent Trade Kit MCP | 官方维护，164 个工具，MIT 开源 |

## 4. 第 3 步 — 策略框架

### 4.1 策略接口规范

所有策略都遵循统一接口，放在 `backend/strategies/` 下：

```python
# backend/strategies/example_ma_cross.py

"""
策略名: 双均线交叉
描述: 快线上穿慢线买入，下穿卖出
参数:
  - fast_period: 快线周期 (默认 5)
  - slow_period: 慢线周期 (默认 20)
"""

class StrategyConfig:
    """策略元信息，前端表单由此生成参数输入框"""
    name = "双均线交叉"
    description = "快线上穿慢线买入，下穿卖出"
    params = {
        "fast_period": {"type": "int", "default": 5, "min": 2, "max": 200, "label": "快线周期"},
        "slow_period": {"type": "int", "default": 20, "min": 5, "max": 500, "label": "慢线周期"},
        "stop_loss_pct": {"type": "float", "default": 0.05, "min": 0.01, "max": 0.5, "label": "止损比例"},
    }


class Strategy:
    """策略逻辑"""

    def __init__(self, **params):
        self.params = params

    def on_bar(self, klines: "pd.DataFrame") -> dict:
        """
        每根 K 线调用一次，返回交易信号

        参数:
            klines: DataFrame，包含 [open, high, low, close, volume]
        返回:
            {
                "action": "buy" | "sell" | "hold",
                "price": float,
                "amount": float,
                "reason": str  # 信号原因
            }
        """
        fast = klines['close'].rolling(self.params['fast_period']).mean()
        slow = klines['close'].rolling(self.params['slow_period']).mean()

        if len(fast) < 2:
            return {"action": "hold", "price": 0, "amount": 0, "reason": "数据不足"}

        if fast.iloc[-2] <= slow.iloc[-2] and fast.iloc[-1] > slow.iloc[-1]:
            return {"action": "buy", "price": klines['close'].iloc[-1],
                    "amount": 1, "reason": "快线上穿慢线（金叉）"}
        elif fast.iloc[-2] >= slow.iloc[-2] and fast.iloc[-1] < slow.iloc[-1]:
            return {"action": "sell", "price": klines['close'].iloc[-1],
                    "amount": 1, "reason": "快线下穿慢线（死叉）"}
        return {"action": "hold", "price": 0, "amount": 0, "reason": ""}
```
