# FreedomForge 版本演进计划

## v0.0.1 — 框架基础版 ✅

> **Tag:** `v0.0.1` | **日期:** 2026-07-15

- FastAPI 后端 + SQLite 数据库
- JWT 认证登录 / 注册 / 登出
- RBAC 权限体系：角色表 → 菜单表 → 关联表
- 用户管理界面（列表、编辑角色、删除）
- 角色管理界面（CRUD + 菜单权限树分配）
- 菜单管理界面（树形 CRUD，支持目录/菜单/按钮三级）
- 动态路由过滤 + 按钮级权限指令
- Naive UI 暗色主题 + 登录页品牌定制
- CLAUDE.md / Makefile / 自动格式化 工程配套

## v0.1.0 — 脚本管理中心（计划中）

**目标：** 真正开始做业务，用户可以在平台上管理和执行 Python 脚本。

| # | 功能 | 说明 |
|---|------|------|
| 1 | 脚本目录浏览 | 读取 `scripts/` 目录，树形展示所有 `.py` 文件 |
| 2 | 脚本在线编辑 | Monaco Editor 或简易代码编辑器，在线修改脚本 |
| 3 | 一键执行 | 前端触发 → 后端 `subprocess` 执行 → 实时返回输出 |
| 4 | 执行历史 | 记录每次执行的时间、耗时、输出日志 |
| 5 | 脚本分类标签 | 用户可为脚本添加标签（如：文件处理/数据采集/量化）|
| 6 | 定时任务 | 基于 APScheduler 或 cron，设置脚本定时执行 |

## v0.2.0 — 自动化中枢（计划中）

**目标：** 从"脚本管理"升级为"事件驱动的自动化引擎"。

| # | 功能 | 说明 |
|---|------|------|
| 1 | 触发器系统 | 文件变动 / 时间到达 / Webhook 接收 → 自动执行脚本 |
| 2 | 任务流编排 | 多脚本按序/并行执行，支持依赖关系 |
| 3 | 通知推送 | 脚本执行完成/失败 → 邮件 / 微信 / Webhook 通知 |
| 4 | 变量与密钥管理 | 集中管理环境变量和 API Key，脚本中引用 |

## v0.3.0 — 量化交易沙盒（计划中）

**目标：** 集成本地量化回测与实时监控。

| # | 功能 | 说明 |
|---|------|------|
| 1 | 数据源管理 | 对接 Tushare / AKShare 等数据源，定时拉取行情 |
| 2 | 回测引擎 | 基于 backtrader / vnpy 的策略回测，前端配置参数 |
| 3 | 策略库 | 策略模板管理 + 在线编辑 + 版本历史 |
| 4 | 回测报告 | 收益率曲线、夏普比率、最大回撤可视化 |
| 5 | 实时信号监控 | 盘中条件触发 → 信号提醒 |

## v1.0.0 — 正式版（计划中）

**目标：** 功能完整、稳定可靠的个人工具平台。

| # | 功能 | 说明 |
|---|------|------|
| 1 | 数据仪表盘 | 脚本执行统计、系统健康状态总览 |

| 4 | 移动端适配 | 响应式优化 + PWA，手机查看执行状态 |
| 5 | 性能优化 | 前端打包优化、后端异步任务队列 |
